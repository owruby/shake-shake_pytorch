# Shake-Shake regularization
PyTorch implementation of shake-shake regularization.

## Dependencies
- python 3.5
- PyTorch 1.0.0

## Accuracy

### CIFAR-10
|Model|Method|This implementaion |Paper|
|:---:|:---:|:---:|:---:|
|ResNet26-2x32|S-S-I|96.37|96.45|
|ResNet26-2x64|S-S-I|97.07|97.02|
